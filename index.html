Summary

The project contains mainly 2 sections

  1. Using Shiny create an interactive application and publish it to shinny apps
  2. Create slides using slidify

Background

This exercise belongs to Data product courses which is part of Data Science specialization from Johns Hopkins University. The data is picked from built in R dataset. Where we are trying to use some interactivity using different controls, like slider, dropdown, submit button & charts.

UI Code Review

hinyUI(pageWithSidebar( headerPanel('Iris k-means clustering'), sidebarPanel( selectInput('xcol', 'X Variable', names(iris)), selectInput('ycol', 'Y Variable', names(iris), selected=names(iris)[[2]]), sliderInput('clusters', 'Cluster count', 3, min = 1, max = 9), submitButton("Go!") ), mainPanel( plotOutput('plot1') ) ))

Server COde Review

shinyServer(function(input, output, session) {

selectedData <- reactive({

iris[, c(input$xcol, input$ycol)]

})

clusters <- reactive({ kmeans(selectedData(), input$clusters) })

output$plot1 <- renderPlot({ par(mar = c(5.1, 4.1, 0, 1)) plot(selectedData(), col = clusters()$cluster, pch = 20, cex = 3) points(clusters()$centers, pch = 4, cex = 4, lwd = 4) })

})
